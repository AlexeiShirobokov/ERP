## Сбросить порт, если он занят

### Посмотреть процесс на порту `8000`

```bash
lsof -i :8000
```

После выполнения команды будет виден процесс, который занимает порт.

### Завершить процесс

```bash
kill -9 PID
```

Например:

```bash
kill -9 1443
```

### Завершить процесс на порту `8000` одной командой

```bash
lsof -ti:8000 | xargs kill -9
```
